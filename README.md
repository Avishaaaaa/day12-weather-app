# 🌤 Weather App

## Project Overview

This project is a simple Weather Application built using HTML, CSS, and JavaScript.

It fetches real-time weather information from the OpenWeatherMap API using Fetch API and Async/Await.

---

## Features

- Search weather by city name
- Displays city name
- Displays temperature
- Displays weather condition
- Displays humidity
- Displays wind speed
- Displays weather icon
- Loading message while fetching data
- Error handling for invalid city names
- Responsive design

## How to Run

1. Download the project.
2. Generate an API Key from OpenWeatherMap.
3. Replace YOUR_API_KEY in script.js.
4. Open index.html in the browser.
5. Search any city.
